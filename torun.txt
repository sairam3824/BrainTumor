Brain Tumor Web App: How To Run
===============================

Files to know:
- app.py
- serve.py
- wsgi.py
- requirements.txt
- WEBAPP.md


Option 1: Conda on macOS
------------------------

1. Open Terminal
2. Go to the project folder:
   cd BrainTumor

3. Create a Conda environment:
   conda create -n brain-tumor-web python=3.11 -y

4. Activate it:
   conda activate brain-tumor-web

5. Install dependencies:
   pip install -r requirements.txt

6. Optional: build cached inference assets first
   python app.py --prepare

   First-time build speed options (--tda-mode):
   python app.py --prepare --tda-mode fast       # ~3 min, H0 only, 110 dims
   python app.py --prepare --tda-mode standard    # ~10 min, H0+H1, 440 dims (default)
   python app.py --prepare --tda-mode full        # ~25 min, H0+H1 + persistence images, 840 dims

   Use "fast" for quick demos, "standard" for normal use, "full" for final submission.
   This only affects the first build. After that, cached assets load instantly.

7. Start the app:
   python app.py
   python app.py --tda-mode fast      (if you want a faster first build)

8. Open in browser:
   http://localhost:8000

9. Production-style local server:
   python serve.py

10. If port 8000 is not available, the app automatically moves to:
    http://localhost:8001

Useful Notes
------------

- First run can take time because the app may build cached assets under:
  output/webapp_artifacts/

- Control first-time build speed with --tda-mode:
  --tda-mode fast       ~3 min   (H0 only, 1 scale, 110 TDA dims, 6 CNN epochs)
  --tda-mode standard   ~10 min  (H0+H1, 2 scales, 440 TDA dims, 8 CNN epochs) [default]
  --tda-mode full       ~25 min  (H0+H1, 2 scales + persistence images, 840 TDA dims, 12 CNN epochs)

- If you only want to prepare the models first:
  python app.py --prepare
  python app.py --prepare --tda-mode fast

- If port 8000 is busy, the app automatically tries:
  http://localhost:8001

- The app pages are:
  Home: http://localhost:8000/
  Analyze: http://localhost:8000/analyze
  History: http://localhost:8000/history

- To stop the app:
  Press Ctrl+C

- If you are already inside an environment, check with:
  python --version
  which python

- On Windows, use this instead of "which python":
  where python
