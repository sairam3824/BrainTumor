# Brain Tumor Analysis Project

This project focuses on identifying brain tumors from MRI scans using deep learning models.

## Structure
- `app.py`: Backend application.
- `brain_tumor_web/`: Frontend interface.
- `requirements.txt`: Python dependencies.
- `notebooke30d8b8ae0-2-1.ipynb`: Analysis and training notebook.

## Getting Started
Ensure all requirements are installed:
```bash
pip install -r requirements.txt
```

Run the application:
```bash
python app.py
```
